Quick Setup — 5 Langkah Mulai

  1. 1

    Setup GitHub Token

    Buka Settings → Masukkan GitHub Personal Access Token untuk publish LP ke repo.

  2. 2

    Setup CAPI & Webhook

    Paste Facebook CAPI Token dan GAS Webhook URL di Settings untuk tracking & order management.

  3. 3

    Buat LP Pertama

    Buka Builder → Upload foto, isi nama produk, harga, benefit, dan konfigurasi lainnya → Publish.

  4. 4

    Test Order

    Buka LP yang sudah dibuat → Isi form order → Cek Dashboard apakah order masuk.

  5. 5

    Go Live!

    Pasang URL LP di Meta Ads → Monitor order di Dashboard → Scale iklan yang profitable.
