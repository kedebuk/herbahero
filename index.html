📋 Cara Publish LP Baru

  1. Setup sekali → Buka Settings → Masukkan GitHub Token
  2. Buka Builder → Isi Nama LP + Slug + Upload foto + Konfigurasi form
  3. Klik "Publish ke Website" → LP langsung live di herbahero.my.id/[slug]
  4. Kelola di LP Manager → Edit, preview, atau hapus LP yang sudah ada
