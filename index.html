📋 Cara Cepat Tambah LP Baru

  1. Pakai Builder → Buka Builder → Isi semua field → Upload foto hero → Klik "Generate & Download"
  2. Upload file HTML → Taruh file yang di-download ke root folder repo ini
  3. Push ke GitHub → File otomatis live di GitHub Pages
  4. Atau manual: Duplikat lp-template.html → Rename → Edit LP_CONFIG di bagian atas
