🔑 GitHub Personal Access Token
Token digunakan untuk publish LP langsung ke GitHub repo. Cara buat:
  1. Buka github.com/settings/tokens/new
  2. Note: Herbahero LP Publisher
  3. Expiration: No expiration (atau pilih durasi)
  4. Centang scope: repo (full control)
  5. Generate token → Copy, paste di bawah

Tersimpan lokal di browser ini, tidak dikirim ke mana-mana.
