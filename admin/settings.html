⚙️ Settings GitHub

🔑 GitHub Personal Access Token
Token digunakan untuk publish LP langsung ke GitHub repo. Cara buat:
  1. Buka github.com/settings/tokens/new
  2. Note: Herbahero LP Publisher
  3. Expiration: No expiration (atau pilih durasi)
  4. Centang scope: repo (full control)
  5. Generate token → Copy, paste di bawah

Tersimpan lokal di browser ini, tidak dikirim ke mana-mana.

☁️ Cloudflare Custom Domain
Opsional — untuk konek LP ke domain sendiri (misal: produkku.com) via Cloudflare Workers. Cara setup:
  1. Buka Cloudflare → API Tokens → Create Token
  2. Pilih template "Edit Cloudflare Workers"
  3. Tambah Zone Resource → Include All Zones (atau pilih domain spesifik)
  4. Copy token → paste di bawah. Account ID ada di sidebar kanan Cloudflare Dashboard.

Tersedia di halaman Cloudflare Dashboard → sidebar kanan
